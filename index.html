hexo搭建个人博客教程

环境：windows
一、 安装node.js环境
浏览器搜索node.js，选择官网打开，下载安装（一直下一步安装就可以）。安装完成以后，打开cmd，node –v，给出node.js的版本，表示安装成功。
二、 注册github
浏览器搜索github，进入官网注册，输入用户名邮箱密码，提交并邮箱验证就可以(地球人都会)。新建一个仓库(repository)，仓库名字是 [yourname].github.io，（注意：yourname是你的用户名，必须是这种形式。）
三、 安装git环境
浏览器搜索git，官网下载安装，一直下一步就好。安装完成后，打开cmd命令行，输入命令git，给出git的版本等相关信息，表示git安装完成
四、 GithubSSH配置
git使用https协议，每次pull,push都要输入密码，使用git协议，使用ssh秘钥，可以省去每次输密码。
SSH配置步骤：
第一步，在git的安装路径下找到git-bash.exe，打开，输入命令：
ssh-keygen –t –rsa –c “你的github注册的邮箱地址”，然后回车回车回车就可以。
解释一下这里弹出的命令的意思，运行命令弹出的第一条是让你输入一个文件名，用于保存刚才生成的SSH key代码，可以不输入文件名，直接回车，用默认的文件名就好
接下来会弹出两次让你输入密码的命令，该密码是push文件时要输入的密码，不是github管理者的密码。这里也可以直接回车，不输入密码，那么push文件时就不需要输入密码，直接提交到github上。
第二步，打开github→点击头像→settings→SSH and KPG keys→New SSHkeys，title部分随便写，key部分，要先在c盘里搜索.ssh，找到id_rsa.pub，用记事本打开，复制所有内容，粘贴到key部分，最后点击ADD SSH key。大功告成。
五、安装hexo
浏览器搜索hexo，点击 文档|hexo 进入官网的文档页面，会有一个安装命令：
nmp install hexo-cli –g
第一步，可以cmd进入dos命令，打开想安装hexo的文件夹，输入上述命令，回车安装hexo（安装略慢）
第二步，安装好以后继续在dos命令界面，继续在这个文件夹下，输入命令：
hexo init myblog （注:myblog是你随便给自己的博客起的名字）
第三步，第二步完成以后，进入博客目录，输入以下命令安装依赖：
nmp install
第四步，安装好以后，就可以本地运行一下；先要部署端口，用以下命令：
hexo s=hexo server (这两个命令等价，启动)
hexo s –p 5555 （这里用的是5555端口，可以改为别的。也可以用hexo s命令，这里是用了默认端口4000）
部署好以后就可以打开浏览器，地址栏输入：
Localhost:5555
打开页面，就能看到自己的博客，官方自带主题。
第五步，安装hexo-deployer-git，输入命令：
nmp install hexo-deployer-git –save 回车安装
六、 安装sublime
浏览器搜索sublime，进入官网下载安装，下一步下一步下一步就好。
安装打开以后，可以直接把刚才创建的博客目录拖进来，sublime很快打开这个文件夹，就可以在这里面修改自己的博客内容，修改相关的配置。
七、将hexo发布至github
在sublime中打开博客的config.yml文件修改相关配置，

author: someody（这里是你起的作者的名字）
url: https://yourname.github.io (注:yourname是你的github用户名)
type:git
Repo:“你的仓库网址”
后面这个网址是你的仓库的网址，获得这个网址的步骤：
登陆github→你的头像→your profile→打开你的仓库→找到clone or download按钮并且点击→然后会给出网址→将得到的网址填到repo:后面
Branch:master 这个默认就好

以上配置好以后（记得要保存），打开cmd命令窗口，进入到自己创建的博客目录下，输入命令：

hexo g =hexo generate（生成静态网站的命令）
hexo d （提交）
首次提交会让你配置github的信息
输入命令：
git config - -global user.email “you@example.com”
git config - -global user.name “yourname”
(这两个命令上面会有提示，只需要修改成自己的邮箱地址和用户名即可)

部署完成以后，会弹出一个让你输入用户名和密码的，输入提交。提交成功以后就可以在浏览器上用域名浏览博客了。域名格式为：url: https://yourname.github.io (注:yourname是你的github用户名)
八、浏览hexo主题网，选一款自己喜欢的主题
将博客主题换掉的步骤如下。
在hexo主题网选一款自己喜欢的主题，然后打开，看一下主题介绍，一般主题介绍里都会有这个主题的下载地址或者安装方法等，下载主题，放到你的博客文件夹里的themes文件夹下，然后在config.yml内的修改theme设定就好了。

注：每次修改完以后要保存，然后在cmd命令界面进入博客文件夹下，输入命令：
hexo g （生成）
hexo d （提交）
然后再配置端口本地访问或者直接远程访问刷新即可。
